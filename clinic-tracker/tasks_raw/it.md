## 2 Months Before Opening
- Schedule internet setup
- Order tech items and arrange delivery
- Confirm internet setup dates with Strategic Growth team

## 1 Month Before Opening
- NextTech setup:
  - Create NextTech elements and clinic ops check
  - Link digital signature to NextTech
  - Generate provider profile in NextTech
  - Access provider's account and add signature as default signature (with date)
  - Change preferences to automatically change status of eMins signed to "Sign by Provider"
  - Add new location to NextTech:
    - If ASC: add location name, add location as ASC location billing entity name
    - If regular location: add name and information to NextTech
- EMR setup: add provider info to ultrasound note

## 1 Week Before Opening

- Designate person to be on call for first day
- Add location to facility/badge access system
